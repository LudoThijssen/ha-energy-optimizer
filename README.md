# HA Energy Optimizer

> Smart home battery optimization for Home Assistant with dynamic energy pricing.

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![Home Assistant](https://img.shields.io/badge/Home%20Assistant-Add--on-41BDF5.svg)](https://www.home-assistant.io/)

## Installation — Method 1: Automatic via HA Add-on Store (recommended)

1. Go to **Settings → Add-ons → Add-on store** in Home Assistant
2. Click **⋮** (top right) → **Repositories**
3. Add this URL:
   ```
   https://github.com/YOUR_USERNAME/ha-energy-optimizer
   ```
4. Find **HA Energy Optimizer** in the store → **Install**
5. Configure via the **Configuration** tab → **Start**
