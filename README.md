## Installation / Installatie

### Automatic / Automatisch

1. Go to **Settings → Add-ons → Add-on store → ⋮ → Repositories**
2. Add: `https://github.com/LudoThijssen/ha-energy-optimizer`
3. Install **HA Energy Optimizer**
